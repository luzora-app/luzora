Why Luzora requests browser permissions

Current Tab

Reads safe metadata from the active page only when Luzora or Bolt is activated.

Notifications

Delivers scheduled task reminders and daily summaries.

Storage

Keeps your settings, tasks, and unfinished drafts locally.

Tabs

Opens saved task links in new tabs when it's time to act.

Luzora accesses the current page only when you invoke it. It never reads passwords, payment information, private form values or your browsing history.
